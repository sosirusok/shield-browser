# 🛡 Shield Browser

가볍고 빠른 **프라이버시 브라우저**. 링크를 붙여넣으면 그 자리에서 사이트가 열리고,
광고·트래커는 기본 차단됩니다. 엔진은 Chromium(Electron)이라 속도는 크롬과 동일합니다.

## 무엇을 막아주나 (덕덕고 스타일)
- **흔적 안 남기기(항상 켜짐)** — 쿠키·방문기록·캐시를 디스크에 저장하지 않고 메모리에만 둠.
  창을 닫으면 모두 사라짐 → 자동 로그인·기록이 노트북에 안 남음(매번 새로 시작).
- **광고 / 트래커 차단** — EasyList + EasyPrivacy 프리빌트 필터 (`@ghostery/adblocker-electron`)
- **스피드다이얼** — 새 탭/첫 화면에 바로가기(네이버·구글·유튜브·챗지피티·제미나이·클로드·그록·인스타그램)
- **제목표시줄 없는 화면** — 화면 공간 최대 활용(창 버튼은 탭바에 통합, 탭바를 끌어 창 이동)
- **권한 기본 거부** — 위치·카메라·마이크·알림을 사이트가 함부로 못 씀
- **WebRTC 로컬 IP 누출 차단** · **DNT / GPC** 헤더 · **깨끗한 UA**(크롬과 동일) · 텔레메트리/구글서비스 off
- **잘 열림** — 모든 사이트 기본 http 로 엶(https 사이트는 서버가 리다이렉트), **QUIC(HTTP/3) 비활성**으로
  학교망 UDP 차단 우회(chess.com 등), 연결 실패는 1회 자동 재시도

## 무엇을 못 하나 (정직하게)
이 브라우저는 **웹사이트·광고사로부터의** 추적을 막고 **이 노트북에 흔적을 안 남깁니다**.
하지만 **기기에 관리자 권한으로 설치된 모니터링/관리 소프트웨어(학교·회사 MDM 등)나
통신사·기관의 망 차단(SNI 차단 등)은 우회하지 않습니다.**
그건 어떤 브라우저도 원리상 불가능하고, 의도하지도 않았습니다. 덕덕고도 마찬가지입니다.

## 실행
```powershell
npm install     # 최초 1회 (Electron + 필터 다운로드)
npm start
```

## 단축키
| 키 | 동작 |
|---|---|
| Ctrl+T | 새 탭 |
| Ctrl+W | 탭 닫기 |
| Ctrl+L | 주소창 포커스 |
| Ctrl+R / F5 | 새로고침 |
| 가운데 클릭(탭) | 탭 닫기 |

## 구조
```
src/main.js      메인 프로세스 — 탭(WebContentsView)·차단·프라이버시 하드닝·IPC
src/preload.js   안전한 contextBridge (window.shield)
ui/              브라우저 크롬 (탭바·주소창·설정 패널)
```

> ※ 검색 엔진은 없습니다. 주소창엔 URL만 입력하세요. (점이 포함된 도메인 또는 http/https 링크)
